Reference: RegistrationKeys.txt

Purpose

Contains all of the registration keys available for account creation.

File Format

List file. One entry per line.
Comment character: semicolon (;)
No header.

Any non-empty, non comment line is added to the list as a key.

Registration keys must not exceed 35 characters. Must be longer than 1 character.

They should be alphanumeric only (a-z, A-Z, 0-9) and should not have any spaces or punctuation.

They can be generated in any fashion. Random characters, words, anything that abides by the above rules.
