Reference: Default Character (0.txt)

Purpose

A required character file for default character stats. This used to have more importance long ago when the server was first being developed, but a few properties are still important.

Most notable is the default quickbar setting (in the preferences section) which is required for the autoattack buttons to work properly.

If a missing or broken character is attached to an account, this is what you see.

For the actual file format, see Character File Format
