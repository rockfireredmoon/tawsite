Reference: Packages (Folder Index)

Purpose

Defines some list files that allows loading of multiple CreatureDef, ItemDef, and Quest files.

Files

CreaturePack.txt - Contains a list of CreatureDef files to load.

ItemPack.txt - Contains a list of ItemDef files to load.

ItemPackOverride.txt - Contains a list of ItemDef files which are loaded, but will specifically replace existing entries rather than add new ones. This is intended to patch original items without modifying the authentic data (as it was extracted from SPM/EER).

QuestPack.txt - Contains a list of Quest files to load.
