Reference: ItemPackOverride.txt

Purpose

Contains a list of ItemDef files which are loaded, but will specifically replace existing entries rather than add new ones. This is intended to patch original items without modifying the authentic data (as it was extracted from SPM/EER).

File names should include the relative path from the base server folder. Paths will be automatically adjusted with the correct backslash or slash if using a Windows or Linux version of the server.

File Format

List file. One entry per line. Comment character: semicolon (;)
No header.
