Reference: QuestPack.txt

Purpose

Contains a list of QuestDef files to load.

File names should include the relative path from the base server folder. Paths will be automatically adjusted with the correct backslash or slash if using a Windows or Linux version of the server.

Files will be loaded in their listed order. Be careful when adding Quest entries, to avoid using an ID that is defined in another file, as this may cause undesired results.

File Format

List file. One entry per line. Comment character: semicolon (;)
No header.
