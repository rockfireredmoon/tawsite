Reference: Instance (Folder Index)

Purpose

Contains additional data files specific to each Zone ID which are loaded whenever a new instance is created. (see ZoneDef.txt for information on Zones and IDs.

There is a folder corresponding to each Zone ID. Within those folders are files that apply to only that zone. If a file does not exist, that zone has no data for that particular feature.

The following files, if present in the folder, may be loaded:

Script.txt - Instance script (triggered spawns, teleports, etc).

EssenceShop.txt - Holds contents of essence chests (any chest that exchanges essences/tokens/keys/etc for items).

Shop.txt - Holds contents of NPC vendor shops.

Static.txt - Holds a list of static NPC objects that are spawned globally (not by spawn points).
