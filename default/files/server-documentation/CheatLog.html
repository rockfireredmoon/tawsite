Reference: CheatLog.log

Purpose

This is a server log file that contains warnings of possible suspect activity.

Botting messages are logged when a player uses too many abilities from a single location. False positives are extremely common, easily generated by players spamming their abilities or during long fights. These are merely warning messages, it is up to the server administrator to verify the locations and actions in game, and how frequently they occur.

Other messages include speed warnings, if the player is moving too fast or is moving at unexpected speeds.

If the file becomes too large, it may be deleted when the server is not running.
