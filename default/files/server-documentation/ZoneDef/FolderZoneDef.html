Reference: ZoneDef (Folder Index)

Purpose

Contains all ZoneDef files, one for each grove. Each file contains a single ZoneDef entry for that grove.

If manually adding groves, a new file with appropriate information will need to be added into the ZoneDef folder. A corresponding entry will also need to be added into Dynamic/ZoneIndex.txt.

Refer to the ZoneDef page for information on the ZoneDef file [ENTRY] format.
